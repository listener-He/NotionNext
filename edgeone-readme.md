# 腾讯云EdgeOne部署配置

## 文件说明

此目录包含用于将NotionNext博客系统部署到腾讯云EdgeOne的配置文件：

- `edgeone-deploy.json` - EdgeOne部署配置文件
- `EDGEONE_DEPLOY.md` - 详细的EdgeOne部署指南

## 部署概述

腾讯云EdgeOne是一个边缘安全加速平台，提供全球CDN加速、安全防护和静态网站托管服务。这些配置文件使您能够将Next.js应用以静态导出模式部署到EdgeOne。

## 主要特性

1. **静态导出**: 将Next.js应用构建为静态HTML文件
2. **RSS重定向**: 配置RSS相关路径的重定向规则
3. **缓存优化**: 为静态资源设置合适的缓存策略
4. **安全头部**: 添加安全相关的HTTP响应头
5. **边缘规则**: 在CDN边缘节点处理URL重写和重定向

## 部署流程

1. 确保项目配置为静态导出模式
2. 在EdgeOne控制台连接代码仓库
3. 使用提供的配置文件进行部署
4. 配置环境变量
5. 添加自定义域名并启用SSL

## 注意事项

- 静态导出模式下，部分服务器端功能可能受限
- API路由需要特别处理以在静态环境中正常工作
- 定期检查部署配置以确保安全性和性能
